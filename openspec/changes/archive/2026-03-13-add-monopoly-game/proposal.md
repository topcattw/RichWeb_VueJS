## 為什麼要做

目前工作區沒有任何前端應用程式，也缺少一個可直接在瀏覽器執行的互動式範例。建立一個以 HTML、Vue Global Build 與 Bootstrap 5 實作的大富翁遊戲，可以同時驗證這套技術架構的可行性，並產出一個可擴充的回合制遊戲基底。

## 變更內容

- 新增一個純前端、免建置工具的單頁大富翁遊戲。
- 提供可設定玩家名稱與人數的開局流程。
- 實作棋盤移動、骰子擲點、經過起點領薪水、購地、收租與破產淘汰等核心規則。
- 提供以 Bootstrap 5 建立的遊戲介面，包含棋盤、玩家狀態、事件紀錄與操作面板。
- 以 Vue 3 Global Build 管理遊戲狀態、計算屬性與 UI 互動，不引入打包流程。

## 能力範圍

### 新增能力
- `game-setup-and-session`: 建立遊戲開局、玩家初始化、回合順序與勝負判定。
- `board-turn-flow`: 管理棋盤格子、骰子移動、起點獎勵、格子事件與回合推進。
- `property-economy`: 管理地產擁有權、購買、租金結算、資金變化與破產淘汰。

### 調整既有能力

無。

## 影響範圍

- 新增前端入口檔案與靜態資源，預計包含 HTML、CSS 與 JavaScript 檔案。
- 不需要後端 API、資料庫或額外 npm 套件。
- 使用 CDN 載入 Vue 3 Global Build 與 Bootstrap 5。
- 影響範圍集中在新建立的遊戲頁面與 OpenSpec 需求文件。
